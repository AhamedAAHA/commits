# Ahamed AAH Portfolio

Personal portfolio website built with React, TypeScript, Tailwind CSS, and Vite.

**Live:** https://ahamed-4e0q.onrender.com

## Development

Requires Node.js 24.x.

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```
nn
The project is configured for Netlify, Vercel, and GitHub Pages deployment.
